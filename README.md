# 📺 YouTube Clone

A modern YouTube clone application built with React Native and Expo, featuring a sleek UI with Tailwind CSS styling.

## 🎯 Overview

This project is a feature-rich YouTube clone that replicates the core functionality and design of YouTube. It's built for cross-platform mobile and web use with modern development practices.

## 🛠️ Tech Stack

- **Frontend Framework:** React Native
- **Build Tool:** Expo
- **Styling:** Tailwind CSS
- **Language:** JavaScript
- **Build System:** Metro Bundler
- **Package Manager:** npm

## 📁 Project Structure

```
YouTube-Clone/
├── api/                 # API integration and backend services
├── components/          # Reusable React components
├── constants/           # App-wide constants and configurations
├── assets/              # Images, icons, and static files
├── App.js              # Main application component
├── index.js            # Application entry point
├── global.css          # Global styles
├── tailwind.config.js  # Tailwind CSS configuration
├── babel.config.js     # Babel transpiler configuration
├── metro.config.js     # Metro bundler configuration
├── eas.json            # Expo Application Services configuration
├── app.json            # App metadata and configuration
└── package.json        # Project dependencies
```

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Expo CLI installed globally

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rabiibenrioui/YouTube-Clone.git
   cd YouTube-Clone
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

4. **Run on your device or emulator:**
   - Press `i` for iOS simulator
   - Press `a` for Android emulator
   - Scan the QR code with Expo Go app on your mobile device

## 📱 Available Scripts

- `npm start` - Start the Expo development server
- `npm run android` - Run on Android emulator/device
- `npm run ios` - Run on iOS simulator
- `npm run web` - Run on web browser
